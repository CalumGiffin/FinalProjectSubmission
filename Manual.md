This prototype is run using Unity Hub and Meta Horizon Link software, on Quest 2 or 3 devices.   
Once in editor with the Quest device linked to the computer, the play button in the editor can be used to enter and use the environment.  
Full install instructions for these are found in the ReadMe.md file.

Once in scene, moving the right hand controller in space will move the robotic arm following the users movement,  
holding down the right controllers middle finger grip on the side can be used to grab and interact with  
pen objects and moving them toward the canvas will being drawing interaction.  
Drawings made can be saved to file by pressing the A button on the right controller at any time.
